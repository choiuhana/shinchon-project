## 페이지별 상세 구성

### 🏠 메인 페이지

#### 섹션 구성
```
1. [Header]
   - 로고
   - 글로벌 네비게이션 메뉴
   - 검색 버튼
   - 로그인/회원가입

2. [Hero Section]
   - 메인 슬라이더 (3~5개 이미지/영상)
   - 메인 카피: "아이의 꿈이 자라는 곳, 신촌몬테소리유치원"
   - CTA 버튼 1: "입학 안내"
   - CTA 버튼 2: "유치원 둘러보기"

3. [Quick Info Cards - 4개 카드]
   - 📝 입학 안내
   - 🍽️ 오늘의 식단
   - 📅 이달의 행사
   - 📢 공지사항 (최신 3개)

4. [About Section]
   - 제목: "신촌몬테소리유치원을 소개합니다"
   - 짧은 소개 텍스트 (2~3줄)
   - 교육 철학 요약 (아이콘 3개)
   - "더 알아보기" 버튼

5. [Education Programs - 3개 카드]
   - 🎯 몬테소리 교육
   - ✝️ 기독교 교육
   - 🌱 생태 교육
   - 각 카드: 이미지 + 제목 + 짧은 설명

6. [Facilities Section]
   - 제목: "안전하고 쾌적한 교육 환경"
   - 시설 사진 갤러리 (6~8장)
   - "시설 더보기" 버튼

7. [Latest News - 2개 컬럼]
   - 왼쪽: 공지사항 최신 5개
   - 오른쪽: 포토 갤러리 (최신 6장, 그리드)

8. [Location & Contact]
   - 왼쪽: 지도
   - 오른쪽: 주소, 전화, 이메일, 운영시간, SNS 링크

9. [Footer]
   - 사이트맵
   - 개인정보처리방침, 이용약관
   - 저작권 정보
```

#### 모바일 버전 차이점
- Hero Section: 슬라이더 높이 축소
- Quick Info Cards: 2x2 그리드 또는 세로 스크롤
- Education Programs: 세로 스크롤
- Latest News: 세로 배치

##### 현재 사이트 참고 (마이그레이션 메모)
- 실제 운영 사이트는 별도 사이드 메뉴 없이 단일 `회원정보 수정` 화면으로 제공되며, 자녀 정보 입력 칸이 동일 화면 내에 최대 3개 노출
- 휴대폰 번호는 `국번-중간-끝` 3개 텍스트박스로 분리되어 있고, 자바스크립트 `reg_check()`가 직접 폼을 검증
- 주소 검색은 `/web/include/zip.php` 팝업을 새 창으로 띄우는 구조로, 신규 시스템 설계 시 내장 모달 또는 API 기반 검색으로 교체 필요

### 예비 학부모 입학 문의 플로우
```
1. 메인 페이지 접속

4. "온라인 상담" 클릭

6. 확인 이메일 수신

8. 방문 상담 진행
9. 원서 접수 (온라인 또는 방문)
10. 입학 확정
```

### 재원생 학부모 일상 사용 플로우
```
1. 메인 페이지 또는 즐겨찾기로 접속

3. 메인에서 "오늘의 식단" 확인
4. "학부모 마당" > "반 소식" 클릭
5. 오늘의 활동 사진 확인
6. 댓글 작성
7. 공지사항에서 가정통신문 확인
8. PDF 다운로드
9. 사이트 종료
```
